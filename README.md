# GuiaPerguntas

Projeto desenvolvido durante um curso de Node.js com o objetivo de praticar conceitos fundamentais de back-end, como rotas, requisições HTTP, integração com banco de dados e renderização no servidor.

O foco principal do projeto foi o aprendizado prático e a consolidação da base em desenvolvimento web.

---

## Sobre o projeto

O GuiaPerguntas é uma aplicação simples de perguntas e respostas, onde é possível:

- Criar perguntas
- Visualizar perguntas cadastradas
- Responder perguntas
- Visualizar respostas relacionadas a cada pergunta

Trata-se de um projeto educacional, desenvolvido com o propósito de reforçar conceitos e não de representar uma aplicação final para produção.

---

## Observação técnica

Este projeto utiliza algumas ferramentas que não refletem necessariamente as abordagens mais modernas adotadas atualmente no mercado, como o uso de EJS para renderização de páginas no servidor.

A escolha dessas tecnologias ocorreu exclusivamente para seguir a proposta do curso e aplicar corretamente os conteúdos apresentados.

Atualmente possuo conhecimento de arquiteturas mais modernas, como:

- Front-end desacoplado
- React e comunicação via JSON

Por isso, este projeto deve ser interpretado como parte do processo de aprendizado.
